bc计算指令

发表于 2018-08-28 | 分类于 linux

内置变量

  • scale：指定运算的小数点位数
  • ibase：指定输入的进制数
  • obase：制定输出的进制数
  • last或者.：最近打印的数字

内置函数

  • length：得到变量的有效数字个数
  • read：得到用户输入
  • scale：得到变量小数点后的位数
  • sqrt：开方运算

程序选项

  • -l：提前加载数学库，将scale设为10
  • -q：安静模式，不输出欢迎信息
  • -v：版本信息
  • -s：使用POSIX标准的bc语言，不是GNU扩展语言

数学函数

  • s(x)：sin函数
  • c(x)：cos函数
  • a(x)：atan函数，计算pi
  • l(x)：ln函数
  • e(x)：e的x次幂

二进制转16进制
先指定16进制的输出，再指定2进制的输入

使用方法

计算pi，小数点后10000位，并显示计算时间
