打印1到N位的数

发表于 2018-10-27 | 分类于 算法与数据结构 ， golang

第一题：

输入一个整数N，打印1到最大的N位数。
输入：1
输出：1,2,3,4,5,6,7,8,9

思路：

  1. 首先将要递归的数据分为“前缀部分:需要处理的部分”
  2. 如果N=3，那么需要打印所有2位数，执行10次循环前缀分别为（0…9）,并且继续递归调用
  3. 因为这个题目是组合，所以没有需要处理的部分
  4. 递归的结束条件为当“N <= 2”

第二题：

对输入的字符串进行全排列。
输入：ABC
输出：ABC ACB BAC BCA CAB CBA

思路：

      ABC
  /    |    \
 ABC   BAC   CAB
 /  \  /  \  /  \
ABC|ACB|BAC|BCA|CAB|CB`A
